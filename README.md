# Voynich Claims Explorer

A searchable database of ~20,000 research claims extracted from [voynich.ninja](https://www.voynich.ninja) forum discussions.

**[View the live site →](https://vosreth.github.io/voynich-claims/)**

## Features

- **Search** claims by keyword
- **Filter** by folio, theory, confidence level, or evidence type
- **Source links** to original forum posts
- ~20,000 claims from 3,500+ threads

## Data

| File | Description |
|------|-------------|
| `data/claims.json` | All claims with sources and theory links |
| `data/theories.json` | ~3,000 theories organized by category |
| `data/folios.json` | Folio → claim count mapping |
